### 作成物

ニコニコ動画風のコメントが流せる動画アプリ

### 機能

・コメント投稿機能（APIよりコメントを取得する機能として代用）

・コメント、コメントリストの表示、非表示の切り替え

・ランダムなコメント装飾機能

### Environment

- Xcode: 12.2
- Swift: 5.3.1

### Configuration

- UI implementation: codeLayout + XIB
- Architecture: MVVM + RxSwift

### Setup

1. Clone the project.

```
$ git clone https://github.com/ikemoti/bejiApp
$ cd CATechChallenge
```

3. Run `make setup` .  
```pod
pod update
```
